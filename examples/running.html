There are two main components to running LBPM simulators. First is understanding how to launch MPI tasks on your system, which depends on the particular implementation of MPI that you are using, as well as other details of the local configuration. The second component is understanding the LBPM input file structure. The examples included provide a basic introduction to working with LBPM.
